\doxysubsection{Detailed Description}
A class that manages a sequence of commands in a home automation system. 

The \mbox{\hyperlink{classMacroRoutine}{Macro\+Routine}} class allows users to create a sequence of commands (e.\+g., turn off all lights, lock all doors) and execute them in order. It can store, add, remove, and execute a list of commands.
