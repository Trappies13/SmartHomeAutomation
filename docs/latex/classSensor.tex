\doxysubsection{Detailed Description}
A class that simulates a sensor in the smart home system. 

The \mbox{\hyperlink{classSensor}{Sensor}} class allows the registration of smart devices and notifies them when an environmental condition is met (e.\+g., motion detected). It uses the observer pattern to notify registered devices.
