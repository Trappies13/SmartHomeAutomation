\doxysubsection{Detailed Description}
A class that represents a smart light in the home automation system. 

The \mbox{\hyperlink{classLight}{Light}} class allows the user to turn the light on or off and provides the current status of the light. It implements the \mbox{\hyperlink{classSmartDevice}{Smart\+Device}} interface.
