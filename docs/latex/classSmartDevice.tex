\doxysubsection{Detailed Description}
An abstract base class representing a smart device in the home automation system. 

The \mbox{\hyperlink{classSmartDevice}{Smart\+Device}} class provides an interface for devices like lights, thermostats, and locks. It includes methods for querying the device status, performing an action, and getting the device type.
