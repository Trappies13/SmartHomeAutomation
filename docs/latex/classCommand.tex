\doxysubsection{Detailed Description}
An abstract class representing a command to be executed on a smart device. 

The \mbox{\hyperlink{classCommand}{Command}} class defines an interface for executing operations on smart devices. It follows the \mbox{\hyperlink{classCommand}{Command}} Design Pattern, encapsulating actions as objects that can be executed.
