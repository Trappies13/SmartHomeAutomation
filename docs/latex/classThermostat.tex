\doxysubsection{Detailed Description}
A class that represents a smart thermostat in the home automation system. 

The \mbox{\hyperlink{classThermostat}{Thermostat}} class allows the user to set and adjust the temperature, as well as query its current temperature. It implements the \mbox{\hyperlink{classSmartDevice}{Smart\+Device}} interface.
