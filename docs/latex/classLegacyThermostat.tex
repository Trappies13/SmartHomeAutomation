\doxysubsection{Detailed Description}
A class that represents a legacy thermostat device. 

The \mbox{\hyperlink{classLegacyThermostat}{Legacy\+Thermostat}} class models an older thermostat device that does not have smart functionality. It allows for setting and getting the temperature, but lacks the advanced features of a smart thermostat.
