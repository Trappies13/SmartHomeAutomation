\doxysubsection{Detailed Description}
An adapter class that integrates a legacy thermostat into the smart home system. 

The \mbox{\hyperlink{classSmartThermostatIntegrator}{Smart\+Thermostat\+Integrator}} class adapts the interface of a \mbox{\hyperlink{classLegacyThermostat}{Legacy\+Thermostat}} to make it compatible with the \mbox{\hyperlink{classSmartDevice}{Smart\+Device}} interface. This allows an old thermostat to be controlled like any other smart device in the system.
