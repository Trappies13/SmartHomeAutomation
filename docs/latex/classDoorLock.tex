\doxysubsection{Detailed Description}
A class that represents a smart door lock in the home automation system. 

The \mbox{\hyperlink{classDoorLock}{Door\+Lock}} class allows the user to lock or unlock a door and provides the status of the door lock. It implements the \mbox{\hyperlink{classSmartDevice}{Smart\+Device}} interface.
