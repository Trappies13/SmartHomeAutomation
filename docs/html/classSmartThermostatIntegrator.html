Detailed Description

An adapter class that integrates a legacy thermostat into the smart home system.

The SmartThermostatIntegrator class adapts the interface of a LegacyThermostat to make it compatible with the SmartDevice interface. This allows an old thermostat to be controlled like any other smart device in the system.
